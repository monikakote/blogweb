🌐 Smart Blog Website

A modern, responsive blog website built using **HTML, CSS, and JavaScript**.
This project allows users to explore blogs, search content, and filter posts by categories in a clean and interactive UI.

---
✨ Features

* 📄 Display blog posts using JavaScript array
* 🔍 Search blog posts in real-time
* 🗂️ Filter blogs by categories (Tech, Life, Education)
* 📖 "Read More" option to view full blog content
* 📱 Fully responsive design (mobile + desktop)
* 🎨 Modern and unique UI design

---
🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---
📂 Project Structure

```
blog-website/
│── index.html
│── style.css
│── script.js
```

---

🚀 How to Run the Project

1. Download or clone the repository
2. Open the project folder in VS Code
3. Open `index.html` in browser
   OR
4. Use Live Server extension for better experience

---

📸 How It Works

* Blogs are stored in a JavaScript array
* Search filters blogs based on title
* Category dropdown filters blog types
* Clicking **Read More** opens full blog in a popup
